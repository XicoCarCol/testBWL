PROYECTO PRUEBA CONOCIMIENTOS

PASOS PARA PODER LEVANTAR EL PROYECTO FRONT-END Y BACK-END, ASI COMO EL SCRIPT PARA BASE DE DATOS POSTGRESQL

1- CORRER EN UNA CONEXION NUEVA EN POSTGRESQL, EL ARCHIVO CON NOMBRE USUARIOS.SQL(INCLUIDO EN LA CARPETA PRINCIPAL)

2- DAR UN CLEAN INSTALL SOBRE EL PROYECTO BACK-END, ESTO PARA QUE ACTUALICE DEPENDENCIAS, POSTERIOR SE PROCEDE A LEVANTAR DICHO PROYECTO
POR ENDE AL MOMENTO DE LEVANTAR SE INSERTA UN USUARIO POR DEFECTO EN ESTE CASO ES admin@admin.com Y CONTRASEÑA admin.

3- EN PROYECTO FRONT-END SE DA UN NPM INSTALL PARA ACTUALIZAR NODE_MODULES, POSTERIOR SE PROCEDE A LEVANTAR PROYECTO CON NG SERVE,
PARA PODER ACCEDER AL PROYECTO SE INGRESA LAS CREDENCIALES, MENCIONADAS EN EL PUNTO ANTERIOR.
